Web Previews for SquidLib demos


The Basic demo, a "Lesson 1" roguelike demo where your '@' character moves where you click in a random dungeon.

The Tsar demo, a slightly-less-basic roguelike demo where your '@' hero bumps into dungeon guards ('Ð¯') to defeat them.

The Dungeon demo, a basic dungeon generation sampler where you can press 'r' to remake the map.

The Dawnlike demo, a graphical roguelike demo where your randomly chosen hero bumps into monsters to obliterate them in a cloud of gore.

The Colorful-GDX demo, a graphical roguelike demo that's almost the same as the Dawnlike demo but uses the colorful-gdx library to change colors.

The Babel Bobble demo, which ciphers text using a mix of natural and random languages (it demos SquidLib's FakeLanguageGen).


Web Previews for Non-SquidLib demos


The Northern Lights Shader demo, which is a nice many-colored flowing shader.

The Water Shader demo, which is an edit on the above shader to make it look a little more like water.

The World Viewer demo, which is a simple 3D sphere textured with a randomly selected map from SquidLib or NASA.
